Mantenha o seu supabase.js atual, sem trocar.

ANTES de testar a V2:
1. Abra o Supabase.
2. Vá em SQL Editor.
3. Rode o arquivo 05_scope_members_and_transactions.sql.

Esse SQL faz 2 coisas importantes:
- permite que membros da mesma casa vejam o nome uns dos outros;
- permite editar/excluir lançamentos dentro da casa, não só os próprios.

Depois:
1. Suba os 4 arquivos novos no GitHub.
2. Faça refresh forçado no navegador com Ctrl + F5.
3. Teste login, sair, visão da casa, visão individual, visão compartilhada, edição e exclusão.
